ModeSet App Wireframes (v1)

These wireframes define the core screens required for the first functional iOS version of ModeSet.
They are intentionally low-fidelity and focus on structure, not visuals.

|                                       |

|            in one tap."               |

|               [ Get Started ]         |
+---------------------------------------+

Icon / color

3. Permissions Flow
